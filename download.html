System Requirements

  • Windows XP (with SP2 or higher), Vista or 7
  • .NET Framework 3.5 (Windows XP & Vista only)
  • XNA Framework 3.1
  • DirectX 9.0c
  • Graphics Card with Shader Model 2 Support
  • Display capable of handling 1280x720 resolution

Installation

Unzip the MeteorSwarmSetup.zip file into a temporary directory & run Setup.exe.
